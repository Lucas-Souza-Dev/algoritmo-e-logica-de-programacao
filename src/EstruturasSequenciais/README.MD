Lista de exercícios focada em lógica básica com entrada, processamento e saída de dados.

**📐 Problema: Terreno**

Calcular a área e o preço de um terreno retangular.

Fórmulas:

Área = largura × comprimento
Preço = área × valor do m²

**📏 Problema: Retângulo**

Calcular:

Área
Perímetro
Diagonal

Fórmulas:

Área = base × altura
Perímetro = 2 × (base + altura)
Diagonal = √(base² + altura²)

**👥 Problema: Idades**

Ler nome e idade de duas pessoas e calcular a média das idades.

Fórmula:

Média = (idade1 + idade2) / 2

**➕ Problema: Soma**

Ler dois números inteiros e mostrar a soma.

Fórmula:

Soma = X + Y

**💰 Problema: Troco**

Calcular o troco de uma compra.

Fórmulas:

Total = preço × quantidade
Troco = dinheiro recebido − total

**⚪ Problema: Círculo**

Calcular a área de um círculo.

Fórmula:

Área = π × r²
π ≈ 3.14159

**💼 Problema: Pagamento**

Calcular o salário de um funcionário.

Fórmula:

Pagamento = valor por hora × horas trabalhadas

**🚗 Problema: Consumo**

Calcular o consumo médio de combustível.

Fórmula:

Consumo = distância / combustível

**📊 Problema: Medidas**

Calcular:

Área do quadrado
Área do triângulo retângulo
Área do trapézio

Fórmulas:

Quadrado = A²
Triângulo = (A × B) / 2
Trapézio = ((A + B) × C) / 2

⏱**️ Problema: Duração**

Converter segundos para horas, minutos e segundos.

Fórmulas:

horas = segundos / 3600
minutos = (segundos % 3600) / 60
segundos = segundos % 60
